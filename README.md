# Iris Flower Classification

## Overview

This project uses Machine Learning techniques to classify Iris flowers into three species:

* Setosa
* Versicolor
* Virginica

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Machine Learning Models

* K-Nearest Neighbors (KNN)
* Random Forest Classifier

## Visualizations

* Species Distribution
* Violin Plot
* Correlation Heatmap
* Pair Plot
* Radar Chart
* Feature Importance Analysis

## Results

The Random Forest model achieved high classification accuracy on the Iris dataset.

## Dataset

The project uses the Iris dataset available in Scikit-Learn.
